# Credits

InfoTerm is inspired by and builds on ideas and implementation work from the open-source project **DeskBuddy** by LextZip.

Original project: <https://github.com/LextZip/Deskbuddy>

The original project provided an important basis for the concept of a small ESP32-based desktop information display with weather, widget and status functionality.

InfoTerm continues this idea with a separate project identity, multilingual support, MQTT extensions, configuration cleanup and a repository structure intended for gradual modularization.

Many thanks to the original DeskBuddy author and contributors for making their work available as open source.
